# CSC864-WordBars

Implemented Text Processing Functions
- Lowercasing
- Noise Removal
- Stemming - Porter Stemming Algorithm
- StopWord Removal

Prerequisites
- node > v12.8.1
- npm > 6.10.2
- Google Chrome > 86.0
- Limit of 100 Queries per day per API key

Install
- npm install
